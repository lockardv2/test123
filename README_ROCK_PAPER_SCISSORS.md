# 가위바위보 게임 🎮

플레이어 vs 컴퓨터 가위바위보 게임입니다.

## 게임 설명

가위바위보는 두 사람이 손 모양으로 승부를 겨루는 게임입니다:
- ✌️ **가위**: 보를 이깁니다
- ✊ **바위**: 가위를 이깁니다
- ✋ **보**: 바위를 이깁니다

## 실행 방법

### 1. Python 버전 (터미널)

```bash
python3 rock_paper_scissors.py
```

**기능:**
- 대화형 콘솔 게임
- 점수 추적
- 여러 라운드 플레이 가능
- 'q'를 입력하여 게임 종료

**사용 예시:**
```
=== 선택하세요 ===
1. 가위
2. 바위
3. 보
q. 게임 종료
==================
선택 (1-3, q): 1

플레이어: 가위
컴퓨터: 보
결과: 플레이어!

========================================
플레이어: 1 | 컴퓨터: 0 | 무승부: 0
========================================
```

### 2. HTML 버전 (웹 브라우저)

웹 브라우저에서 `rock_paper_scissors.html` 파일을 열기만 하면 됩니다.

```bash
# Chrome
google-chrome rock_paper_scissors.html

# Firefox
firefox rock_paper_scissors.html

# 또는 파일 탐색기에서 더블 클릭
```

**기능:**
- 아름다운 그래픽 인터페이스
- 버튼 클릭으로 간편한 플레이
- 실시간 점수 표시
- 반응형 디자인 (모바일 지원)
- 게임 초기화 버튼

## 요구 사항

### Python 버전
- Python 3.x 이상
- 추가 패키지 설치 불필요 (표준 라이브러리만 사용)

### HTML 버전
- 모던 웹 브라우저 (Chrome, Firefox, Safari, Edge 등)
- 인터넷 연결 불필요 (오프라인에서 실행 가능)

## 게임 규칙

1. 플레이어가 가위, 바위, 보 중 하나를 선택합니다
2. 컴퓨터가 무작위로 선택합니다
3. 승자가 결정됩니다:
   - 가위 vs 보 → 가위 승
   - 바위 vs 가위 → 바위 승
   - 보 vs 바위 → 보 승
   - 같은 선택 → 무승부
4. 점수가 업데이트됩니다
